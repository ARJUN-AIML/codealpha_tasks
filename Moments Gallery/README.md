About the Project

Moments Gallery is a simple and responsive image gallery created using HTML, CSS, and JavaScript.
The project focuses on presenting images in a clean layout while allowing users to interact with them through category filters and a lightbox view.

This project was built to practice front-end layout design, basic JavaScript functionality, and responsive web design concepts.

What This Project Does

Displays images in a grid-based gallery

Allows users to filter images by category

Opens images in a lightbox view when clicked

Provides next and previous navigation inside the lightbox

Adapts smoothly to different screen sizes

Responsive Design

The gallery layout automatically adjusts based on the device:

Large screens show multiple images per row

Tablets show fewer columns

Mobile devices display one image per row

This ensures the gallery remains easy to use on all screen sizes.
